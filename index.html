Caso de estudio

SmartBudget: Landing Page con SASS y Bootstrap

Descripcion del proyecto

Desarrollo de la landing page completa de SmartBudget, una app de finanzas personales. El sitio incluye navbar responsiva, hero con call to action, grilla de características, sección de pasos, testimonios, planes de precios y un modal de registro. Todo maquetado desde un prototipo en Figma.

Desafio principal

Organizar los estilos de un sitio con múltiples secciones y componentes sin que el CSS se volviera inmanejable. El reto fue aplicar una arquitectura de estilos escalable usando SASS con el patrón 7-1 y la metodología BEM, manteniendo coherencia visual en todas las secciones y responsividad en móvil, tablet y escritorio.

Solucion propuesta

Se estructuró el CSS con SASS siguiendo el patrón 7-1 (abstracts, base, components, layout, pages, themes, vendors). Se aplicó BEM para nombrar las clases de forma predecible. Bootstrap 4 se usó para el sistema de grilla, navbar colapsable y el modal, evitando sobreescribir sus estilos con clases personalizadas.

Herramienta tecnica utilizada

HTML5 semántico, SASS compilado a CSS, Bootstrap 4 via CDN e iconos SVG de Tabler Icons. El prototipo fue desarrollado en Figma antes de codificar. Git para control de versiones y GitHub Pages para el despliegue del resultado final.

Principales aprendizajes

  • Arquitectura SASS con patrón 7-1 y estructura por carpetas
  • Variables, mixins y anidamiento en SASS
  • Metodología BEM aplicada a un proyecto real
  • Integración de Bootstrap 4 sin romper estilos propios
  • Flujo de trabajo Figma a código: prototipo, maquetado, ajuste

Habilidades tecnicas aplicadas

  • HTML5 semántico con jerarquía de componentes
  • SASS: variables, mixins, nesting, partials
  • Metodología BEM para clases CSS
  • Bootstrap 4: grid, navbar, cards, modal
  • Flexbox y media queries para responsividad
  • Git y GitHub Pages para versión y despliegue

Por que elegí este proyecto

SmartBudget es el proyecto donde el salto técnico fue más visible. Pasar de escribir CSS plano a estructurar estilos con SASS y BEM cambió completamente mi forma de trabajar. Aprendí a pensar en componentes antes de escribir una sola línea de código, a usar Bootstrap sin perder el control del diseño y a mantener un proyecto organizado a medida que crece. Es el trabajo que mejor refleja mi avance en desarrollo frontend hasta el momento.
